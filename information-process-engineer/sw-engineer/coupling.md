# 결합도

## 초성으로 외우기

- 내공외제스자

### 내용 결합(강한 결합)

- 다른 모듈의 변수나 기능을 다른 모듈에서 사용하는 경우

### 공통 결합

- 전역 변수를 참조, 공유하는 상태의 경우

### 외부 결합

- 외부에서 도입된 데이터 포맷, 통신 프로토콜을 공유할 경우

### 제어 결합

- 어떻게 해야하는지에 대한 정보를 넘겨 다른 모델의 흐름을 제어

### 스탬프 결합

- 모듈 간 데이터 구조 등이 전달되는 경우

### 자료 결합(약한 결합)

- 모듈 간 파라미터를 통해 데이터를 공유하는 경우
